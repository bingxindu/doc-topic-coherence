Jar built from modified Palmetto code, plus jars containing Plametto dependencies.
Palmetto (not including the depdendancy jars) is licenced under GNU Affero General Public License.
Palmetto_mod.zip contains the modified version of the source code. 

Modification: added user-friendly interface for building coherence functions,
in order to create coherence functions that use custom lucene indices.

This jar has to be wrapped as python package using the jcc tool.
To do this, first run prepare-jars, then jcc-palmetto
